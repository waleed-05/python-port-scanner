# python-port-scanner
A simple multithreaded port scanner built using python sockets.

# Python Port Scanner

## Features
- Scans a range of ports
- Uses threading for faster performance
- Detects open ports

## Usage
python3 scanner.py <target_ip> <start_port> <end_port>

Example:
python3 scanner.py 127.0.0.1 20 100

## Requirements
- Python 3
